Dice Roller Project

REACTJS, DYNAMIC CONTENT, HTML/CSS, RESPONSIVE DESIGN, VECTOR DESIGN

The idea

I made this project to practice setting up a one page app in ReactJS.
I chose to make a dice roller because I play some rpg and often friends or myself forget the dice.
This is an helpful little tool for those situations when you really need a dice but you don't have one.

The execution

The app is structured with 1 statefull and 3 stateless components.
In the components I used render props and I also passed a method.
Two of the components are compiled as lists (the dice and the result of the roll).

The main event handler returns onClick the result of the chocen dice. Because I also put a coin I had to include some conditional rendering for a heads/tails instead of a numeric result.

I imported my own designs for the dice and the logo. I use affinity designer as a designing tool, it allows to realise both vector and pixel based illustrations.

I hosted the dice roller project on Firebase to practice with another hosting provider than Github
